# Outputs for Clearing Non Value Messages

Updated On 22 March 2025 |
 2 Min(s) read

Feedback
Summarize

Account reporting request related enquiries help the user to view the status and details of the sent request.

[Enquiries and Outputs](#)

The user can access the following enquiries

**View Account Reporting Request**

The user can access the enquiry from: User menu > Payments > Payment Hub > Payment Inquiries > Reports > Accounting Report Request >Account Report Request > View Accounting.

This enquiry displays the account reporting request records with the following icons against each of the created records,

1. View Report Request Details – By clicking this icon record view of the sent account reporting request message is displayed.
2. View Outward Request – By clicking this icon user will be displayed with following icons (Message Type, Sent Date and Time, Message Content) displaying details about the sent account reporting request message
3. View Report Details – By clicking this icon record view of the received account report will be displayed
4. View Inward Report - By clicking this icon user will be displayed with following icons (Message Type, Received Date and Time, Message Content) displaying details about the received positive response message
5. View Acknowledgement – By clicking this icon user will be displayed with following icons (Message Type, Received Date and Time, Message Content) displaying details about the received negative response message

**View Message Retrieval Request**:

Bank user can view the sent message retrieval request and its status transition in an enquiry **View Message Retrieval Request**.

**Navigation**: User Menu > Payments > Payment Hub > Investigations & Cancellations > Retrieval Request > US > View Message Retrieval Request.

This enquiry displays the message retrieval request records with the following icons against each of the created records.
