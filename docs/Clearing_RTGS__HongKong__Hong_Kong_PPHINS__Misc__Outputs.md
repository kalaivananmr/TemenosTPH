## Enquiry and Reports

The user can view the below enquiry from this feature:

[Pending and Process Payments](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Payments Enquiry - Transaction wise**

This enquiry displays the list of transaction statuses which are processed and pending to be processed along with View, ViewInDetail and Save to Template options.

- **View** – Clicking this icon displays the transaction details from the order entry screen in read-only mode.
- **ViewInDetail** – Clicking this icon displays the Payment Information screen with multiple options to view the detailed information about the payment.

- **Fees Applied** – Selecting this option and clicking the **Launch** icon displays the details about the Fees applied on the selected payment.
- **Posting Lines** – Selecting this option and clicking the **Launch** icon displays the details about the Posting lines.

- **Save to Template** - Clicking this icon is Saves the payment as template in Order Entry. This will be applicable only for the Payments in status 999.
