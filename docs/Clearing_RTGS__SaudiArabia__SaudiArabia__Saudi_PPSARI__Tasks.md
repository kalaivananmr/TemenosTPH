# Tasks for Saudi Arabian Riyal Interbank Express (SARIE)

Updated On 07 August 2023 |
 2 Min(s) read

Feedback
Summarize

The Saudi Arabian Riyal Interbank Express (SARIE) provides the mechanism for all the Saudi commercial banks to make and settle payments in Saudi Riyals (SR). SARIE provides the basis for improved banking products and services.

## Workflow

This section helps the user to perform the below tasks:

[Initiating SARIE Payments](#)

When a user initiate a transaction using the Payment Order – SARIE menu, the SARIE Credit transfer is initiated from TPH and is sent to clearing.

To initiate a SARIE transaction, follow the below steps:

1. Log in as **PAYUSER1** > **Payment Order** > **Input Payment Order** > **Saudi Arabia** > **SARIE RTGS**.
2. Enter values in the following fields:
   - *Debit Account*
   - *Payment Currency*
   - *Payment Amount*
   - *Beneficiary IBAN*
   - *Beneficiary Name*
   - *Payment Execution Date*
   - *Beneficiary’s Bank BIC*
3. Click the **Validate** icon to check for errors and overrides.
4. After the successful validation, click the **Commit** icon.

[Authorising Payment Order Requests](#)

Payments initiated through Payment Order screen must be authorised.

To authorise a transaction, follow the below steps:

1. Log in as **PAYUSER1** > **Payment Order** > **Authorise/Delete Payment Order** > **Authorise/Delete Payment Order**.
2. Click the **Authorise** icon.
3. Verify the details and click the **Authorise** icon.

The transaction is authorised, if there are no error messages.
