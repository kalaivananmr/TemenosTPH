Lebanon RTGS payment feature related enquiries help the user to view the status and details of the payment.

## Enquiries and Reports

In Lebanon RTGS, the user can view the following enquiries and reports:

[Pending and Processed Payments](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Pending and Processed Payments** > **Payments Enquiry - Transaction wise**

This enquiry displays the list of transactions, which are processed and pending to be processed along with status code. The following options are available for the user to view the transactions:

- **View** – By clicking this icon, the transaction details are displayed.
- **ViewInDetail** – By clicking this icon, the Payment Information screen with multiple options to view the detailed information about the payment is displayed.
