This section helps the user to configure the important parameters relevant to Sri Lanka RTGS.

## `PO` Application for Payment Capture

1. To configure payment order product, go to **Admin Menu**>**Payments Hub**>**Payment Order Admin Menu**>**Payment Order Product** **Setup** (one each for Bank Transfer and Customer Transfer).
2. Enter the required details in the following fields:

| Field | Description |
| --- | --- |
| *ID of the Product* | `PO` application product name |
| *Description* | Sri Lanka RTGS |
| *Channel* | Sri Lanka RTGS channel setup in TPH  - Set as ’*LKB*’ |
| *Pay Through Beneficiary* | Payment is initiated for the biller through a beneficiary ID  - Set as *Yes* |

## Clearing Cut-Off

To configure different cut-off times for Sri Lanka RTGS payments for MT103 or MT202, go to **Routing & Settlement**>**Channel Cut-off.**

## Client Conditions for Separate Charge Booking

To separately book the charges in the statements, link default client condition record to the Sri Lanka RTGS payment product.
