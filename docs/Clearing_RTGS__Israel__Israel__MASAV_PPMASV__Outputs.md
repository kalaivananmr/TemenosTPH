The user can view the below list of enquiries and reports pertaining to MASAV Clearing in the core banking system.

## Enquiries and Reports

The user can access the following enquiries and reports:

[Pending and Processed Payments](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Pending and Processed Payments** > **Payments Enquiry – Transaction wise**

This enquiry displays the list of transaction status details that are processed and pending to be processed along with the following options. Users can view the details by clicking the below icons against each record.

- **View** – Displays the transaction details from the order entry screen.
- **ViewInDetail** – Displays the Payment Information screen with multiple options to view the detailed information about the payment.
- **Save to Template** – Saves the payment is as a template in Order Entry. This is applicable only for the Payments in 999 status.
