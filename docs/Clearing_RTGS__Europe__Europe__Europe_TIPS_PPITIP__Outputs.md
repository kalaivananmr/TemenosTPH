# Outputs for Target Instant Payment Settlement (TIPS)

Updated On 10 February 2021 |
 2 Min(s) read

Feedback
Summarize

TIPS feature related enquiries help the user to view the status and details of the payment.

## Enquiries and Reports

The user can view the below enquiries and reports:

[Pending and Processed Payments](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Pending and Processed Payments** > **Payments Enquiry - Transaction wise**

This enquiry displays the list of payment transactions, which are processed and pending to be processed along with the status code. The following options are available for the user to view the transactions.

- **View** – By clicking this icon, the transaction details are displayed.
- **ViewInDetail** – By clicking this icon, the Payment Information screen with multiple options to view the detailed information about the payment is displayed.

[TIPS Clearing SOA Report](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Reports** > **Local Clearing Reports** > **TIPS Clearing SOA Report**

This enquiry displays the local clearing reports for a particular day, based on company with **View Cos** option. By clicking the **View Cos** icon, transaction details in the report are displayed.
