## Introduction to Roles

The Model Bank is available with pre-configured roles based on the standard organisational structure of a bank or a financial institution, centered on customer services. These are available in the base implementation pack to support faster implementation.

The demo system has been updated with required role-based user IDs and pre-configured passwords, which are provided in the individual roles section.

For example, to login as a Customer Service Agent in the demo system, users can use the user ID CSAGENT and the password 123456.

These user IDs and passwords are available only in the demo system. The clients can review this setup from the demo system and create the credentials for their end users.
