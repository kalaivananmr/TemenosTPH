The NEFT Clearing module related enquiries help the user to view pending and process payments.

## Enquiries and Reports

The following enquiries are available for the NEFT Clearing module.

[Pending and Process Payments](#)

The user can access the PAY.ENQ.REPAIR.NEFT.IN enquiry from the **PAYUSER1 > Payment Hub > Payment Inquiries > Pending and Process Payments – Transaction wise** menu.

This enquiry displays the list of transaction status which are processed and pending to be processed along with the **View** and **View in Detail** options.

- Click on the **View** icon to display the transaction details from the Order Entry screen in read-only mode.
- Click on the **View in Detail** icon to display the Payment Information screen with multiple options to view the detailed information about the payment.
- Click on the **Save to Template** option to save the template in the order entry. This will be applicable only for the payments in status 999.
