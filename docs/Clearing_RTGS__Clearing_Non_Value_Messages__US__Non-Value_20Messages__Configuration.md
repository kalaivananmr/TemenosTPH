[Archival of Accounting Report Request](#)

The ARCHIVE application allows users to set the retention period after which the (records available in the account reporting request application) to be moved from history to archive.

1. Configure the ARCHIVE application as per below.
2. For the Account Reporting Request application, Set the Retention Period to 03M. The records older than the retention period will then be archived.
