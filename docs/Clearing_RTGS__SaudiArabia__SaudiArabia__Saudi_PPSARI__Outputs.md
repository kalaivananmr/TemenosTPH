This topic helps the user to view the enquiries and reports pertaining to Saudi Arabian Riyal Interbank Express (SARIE) feature.

## Enquiries and Reports

This section allows the user to view the enquiries and reports:

[Pending and Process Payments](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Pending and Process Payments – Transaction wise**

This enquiry displays the list of transaction statuses, which are processed and pending to be processed with **View**, **ViewInDetail** and **Save to Template** options.

- **View** – By clicking this icon, displays the transaction details from the order entry screen in view mode.
- **ViewInDetail** – By clicking this icon, displays the Payment Information screen with multiple options to view the detailed information about the payment.
- **Save to Template** - This option is used to save the template in Order Entry and it is applicable only for the Payments in 999 status.
