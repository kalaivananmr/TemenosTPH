BPAY Clearing Australia related enquiries help the user to view the status and details of the payment.

## Enquiries and Reports

In BPAY Clearing Australia, the user can view the following enquiry and report:

[Pending and Processed Payments](#)

The user can access the enquiry from: **PAYUSER1** > **Payment Hub** > **Payment Inquiries** > **Payments Enquiry - Transaction wise**

This enquiry displays the list of transactions, which are processed and pending to be processed along with their status codes. The user has the following options to view the transactions.

- **View** – By clicking this icon, the transaction details from the order entry screen is displayed in view mode.
- **ViewInDetail** – By clicking this icon, the Payment Information screen with multiple options to view the detailed information about the payment is displayed.
